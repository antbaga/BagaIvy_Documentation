# Objects Data (Simulation)

The **Objects Data** panel serves as the central hub for managing the collections used.

---

## Collections

* **Targets**: Displays the collection containing the meshes the ivy grows on.
* **Start Points**: Displays the collection containing your growth origin points.
* **Leaves & Flowers Collections**: Shows which collection are currently assigned to the ivy.

---

## Single Object vs. Collection
By default, the generator uses entire collections, click object icon next to the collection icon to switch single object.

---

!!! tip "Fixing Asset Orientation"
    If your leaves facing the wrong way, use the local rotation in this Section to align them with the branch.
