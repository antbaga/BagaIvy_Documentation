Installation

This guide will walk you through the steps to install Baga Ivy Generator in Blender. The process follows the standard Blender addon installation method.

1. Download the Addon

After your purchase, download the .zip file corresponding to your Blender version. The file should be named:
Baga_Ivy_Generator_VX.X.X_Blender_XX.zip

⚠️ Important: Do NOT unzip the file! > Blender installs the addon directly from the compressed ZIP file. If you extract it manually, the installation might fail or cause errors.

2. Install in Blender

  1. Open Blender.
  2. Go to Edit > Preferences.
  3. Select the Add-ons tab on the left menu.
  4. Click the Install... button at the top right.
  5. Navigate to the folder where you downloaded the .zip file.
  6. Select Baga_Ivy_Generator_VX.X.X_Blender_XX.zip and click Install Add-on.

3. Activate the Addon

Once installed, the list should automatically filter to show Baga Ivy.

  1. Check the box next to Baga Ivy Generator to enable it.
  2. The addon is now active!
  3. Don't forget to save the addon preferences (bottom left button).

Note: If it doesn't appear, type "Baga" in the search bar at the top right of the Preferences window.

4. Where to find it?

The addon is located in the 3D Viewport Side Panel (N-Panel).

In the 3D Viewport, press the N key on your keyboard to open the side panel. Look for the tab named BagaIvy.

How to Update?

If you are installing a new version (e.g., updating from V1.0 to V2.0), you must remove the previous version first to avoid conflicts.

  1. Go to Edit > Preferences > Add-ons.
  2. Search for "Baga".
  3. Uncheck the box to disable the addon.
  4. Click the Remove button to uninstall it completely.
  5. Restart Blender (Crucial step to clear the cache).
  6. Install the new .zip file following the standard procedure.
